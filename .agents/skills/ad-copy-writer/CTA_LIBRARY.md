## CTA A/B Test Ideen

| Test | Variation A | Variation B |
|------|-------------|-------------|
| Verb | "Testen" | "Starten" |
| Timeframe | "Jetzt" | "Heute noch" |
| Benefit | "Kostenlos testen" | "Gratis starten" |
| Urgency | ohne Dringlichkeit | "Nur noch heute" |
| Social | neutral | "+ 2.000 andere" |
| Safety | ohne Garantie | "Risikofrei testen" |

## CTA Don'ts

### Zu vage
```
❌ Hier klicken
❌ Weiter
❌ Mehr
❌ Los
```

### Zu aggressiv
```
❌ JETZT KAUFEN!!!
❌ Letzte Chance EVER
❌ Du MUSST das haben
```

### Zu lang
```
❌ Klicke hier um mehr über unser Angebot zu erfahren
❌ Erhalte jetzt deinen kostenlosen Download sofort
```

### Nicht zur Phase passend
```
❌ "Jetzt kaufen" bei Awareness-Kampagne
❌ "Mehr erfahren" bei Cart-Abandonment
```
