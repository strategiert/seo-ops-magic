# Follow-Up Email Templates

## Follow-Up Strategie

```
Timing:
Tag 1: Initial Pitch
Tag 3-4: Follow-Up 1 (Value Add)
Tag 7-8: Follow-Up 2 (Social Proof)
Tag 14: Final Follow-Up (Graceful Close)

Regel: Max 3 Follow-Ups, dann Pause (min. 3 Monate)
```

---

## Follow-Up 1: Value Add

### Timing: 3-4 Tage nach Initial Pitch

### Ziel: Zusätzlichen Mehrwert bieten

### Template A: Neue Information

```
Subject: Re: [Original Subject] - neue Daten

Hi [Vorname],

kurzer Nachtrag zu meiner Email von [Tag]:

Seit dem hat sich [neue Entwicklung]:
- [Neue Info 1]
- [Neue Info 2]

Falls Sie an [Thema] arbeiten, könnte das
relevant sein.

[Original Angebot] steht weiterhin.

Beste Grüße,
[Name]
```

### Template B: Ressource Teilen

```
Subject: Re: [Original Subject] + Ressource

Hi [Vorname],

ich wollte [Ressource] teilen, die für Ihre
Berichterstattung über [Beat] interessant sein könnte:

[Link oder kurze Beschreibung]

Unabhängig von meinem Pitch - vielleicht nützlich
für Ihren nächsten Artikel.

Falls Sie noch Fragen zu [Original-Thema] haben,
bin ich gerne da.

Beste Grüße,
[Name]
```

### Template C: Alternativer Angle

```
Subject: Re: [Original Subject] - anderer Blickwinkel

Hi [Vorname],

nach nochmaligem Nachdenken:

Falls [Original-Angle] nicht passt, wäre vielleicht
[Alternativer Angle] interessanter?

[1-2 Sätze zum alternativen Angle]

Nur ein Gedanke - kein Druck.

Beste Grüße,
[Name]
```

## Follow-Up 2: Social Proof

### Timing: 7-8 Tage nach Initial Pitch

### Ziel: FOMO / Validierung durch andere

### Template A: Other Coverage

```
Subject: Re: [Original Subject] - Update

Hi [Vorname],

kurzes Update: [andere Publikation] hat gerade
über [Thema] berichtet.

Artikel hier: [Link]

Falls Sie einen eigenen Angle für [Ihre Publikation]
suchen, kann ich [exklusiven Aspekt] anbieten.

Immer noch interessiert an einem Gespräch?

Beste Grüße,
[Name]
```

### Template B: Event/Momentum

```
Subject: Re: [Original Subject] - Momentum

Hi [Vorname],

seit meinem letzten Kontakt ist einiges passiert:

- [Event/Entwicklung 1]
- [Event/Entwicklung 2]
- [Anzahl] Medien haben bereits berichtet

Die Story gewinnt an Fahrt. Falls Sie noch
einsteigen möchten - [Exklusiv-Angebot].

Lassen Sie es mich wissen.

Beste Grüße,
[Name]
```

### Template C: Testimonial/Quote

```
Subject: Re: [Original Subject] - O-Ton

Hi [Vorname],

ich habe [interessantes Zitat/Statement] von
[Person] bekommen, das die Story ergänzen könnte:

"[Zitat]"

Falls Sie an [Thema] arbeiten, könnte das
einen guten Aufhänger bieten.

Interesse?

Beste Grüße,
[Name]
```

## Follow-Up 3: Final / Graceful Close

### Timing: 14 Tage nach Initial Pitch

### Ziel: Respektvoll abschließen, Tür offen lassen

### Template A: Closing the Loop

```
Subject: Re: [Original Subject] - Schließe den Loop

Hi [Vorname],

ich möchte nicht nerven - daher letzter Check:

Ist [Thema] aktuell relevant für Sie?

Falls nicht, kein Problem. Ich halte Sie gerne
auf dem Radar für zukünftige Stories zu [Beat].

Falls ja, stehe ich weiterhin für ein Gespräch
zur Verfügung.

Beste Grüße,
[Name]
```

### Template B: Future-Focused

```
Subject: Re: [Original Subject] - Für später

Hi [Vorname],

ich verstehe, dass Timing oft alles ist.

Falls [Thema] gerade nicht passt, halte ich
Sie gerne für zukünftige Updates auf dem Laufenden:

- [Kommender Meilenstein]
- [Geplante News]
- [Relevanter Event]

Nur kurz Bescheid geben, was Sie interessiert.

Beste Grüße,
[Name]
```

### Template C: Direct Ask

```
Subject: Re: [Original Subject] - Kurze Frage

Hi [Vorname],

eine direkte Frage: Soll ich aufhören,
Sie zu diesem Thema zu kontaktieren?

Kein Problem, falls ja. Ich weiß, dass
Journalisten viele Pitches bekommen.

Falls nein, und es nur am Timing liegt -
wann wäre besser?

Danke für Ihre Zeit.

Beste Grüße,
[Name]
```

## Spezielle Follow-Ups

### Nach Öffnung ohne Reply

```
Subject: Re: [Original Subject]

Hi [Vorname],

ich sehe, dass Sie meine letzte Email geöffnet haben -
danke für Ihre Zeit.

Falls Sie noch Infos brauchen oder Fragen haben,
bin ich gerne da.

Kurz: [1 Satz Zusammenfassung des Angebots]

Beste Grüße,
[Name]
```

### Nach positiver Reaktion

```
Subject: Re: [Original Subject] - Nächste Schritte

Hi [Vorname],

freut mich, dass [Thema] interessant ist!

Hier die nächsten Schritte:
1. [Nächster Schritt 1]
2. [Nächster Schritt 2]

Wann passt es für Sie? Ich bin [Zeitfenster] verfügbar.

Assets sende ich nach Ihrer Bestätigung.

Beste Grüße,
[Name]
```

### Nach "Nicht jetzt" Antwort

```
Subject: Re: [Original Subject]

Hi [Vorname],

verstanden - Timing ist alles.

Darf ich mich in [Zeitraum] nochmal melden?
Oder: Gibt es ein Thema, das Sie gerade
eher interessiert?

Danke für die Rückmeldung!

Beste Grüße,
[Name]
```

## Follow-Up Regeln

### DO ✅

```
✅ Neuen Mehrwert in jedem Follow-Up
✅ Kürzer als der vorherige
✅ "Re:" im Subject behalten
✅ Auf Mobile-Lesbarkeit achten
✅ Respektvoll und professionell bleiben
```

### DON'T ❌

```
❌ "Checking in" ohne Value
❌ "Did you see my email?"
❌ Guilt-Tripping ("Sie antworten nicht...")
❌ Mehr als 3 Follow-Ups
❌ Follow-Up am selben Tag
❌ Identischen Text wiederholen
```

### Timing-Regeln

```
Beste Tage: Dienstag, Mittwoch, Donnerstag
Beste Zeit: 9:00-10:00 Uhr
Vermeiden: Montag morgen, Freitag nachmittag
Nie: Wochenende, Feiertage, Urlaubszeit (Aug, Dez)
```

## Tracking

### Was tracken?

```
- Öffnungsrate pro Follow-Up
- Reply-Rate pro Sequenz-Position
- Beste Converting Follow-Up
- Opt-Out-Rate
```

### Optimierung

```
A/B-Test:
- Subject Lines
- Timing (3 vs 4 Tage)
- Value-Add-Typen
- CTA-Formulierungen
```
