## Wichtige Unterscheidungen

### Immer übersetzen (Deutsch Markt)

```
Data privacy → Datenschutz (nie "Daten-Privatsphäre")
GDPR → DSGVO
Terms of service → AGB / Nutzungsbedingungen
Privacy policy → Datenschutzerklärung
Cookie consent → Cookie-Einwilligung
```

### Anglizismus behalten (etabliert)

```
Marketing, SEO, CRM, KPI, ROI, B2B, B2C,
Landing Page, Lead, Conversion, Funnel,
Content, Traffic, Engagement, Targeting
```

### Kontext-abhängig

```
Feature:
- Produktbeschreibung: "Funktion"
- Tech-Kontext: "Feature"

Interface:
- Allgemein: "Schnittstelle"
- UI: "Benutzeroberfläche"

Content:
- Allgemein: "Inhalte"
- Marketing-Kontext: "Content"
```
