# Pitch Email Templates

## Grundstruktur

```
Subject: [Newsworthy Hook + Personalisierung]

Hi [Vorname],

[Hook: 1 Satz, der Aufmerksamkeit erregt]

[Kontext: Warum ich Sie kontaktiere - mit Personalisierung]

[News/Story: 2-3 Sätze, was die Story ist]

[Value Prop: Was Sie exklusiv bekommen]

[CTA: Klare nächste Schritte]

Beste Grüße,
[Name]
[Titel]
[Kontakt]
```

---

## Template 1: Produkt-Launch

### Subject Lines

```
Option A: "[Publikation] Exklusiv: Erste [Innovation] für [Markt]"
Option B: "Neue [Kategorie]-Lösung löst [Problem] - Exklusiv-Zugang"
Option C: "Re: Ihre Berichterstattung über [verwandtes Thema]"
```

### Email Body

```
Hi [Vorname],

[Personalisierungs-Hook: Ihr Artikel über X hat mich angesprochen]

Kurz: Wir launchen [Produktname], die erste [Innovation] für [Zielmarkt].

Das Besondere:
- [USP 1 - mit Zahlen wenn möglich]
- [USP 2]
- [Problem, das gelöst wird]

Hintergrund: [1 Satz zur Gründung/Team/Funding]

Ich würde Ihnen gerne vorab Zugang + ein Gespräch mit unserem
CEO [Name] anbieten - vor dem offiziellen Launch am [Datum].

Hätten Sie diese Woche Zeit für ein kurzes Telefonat?

Beste Grüße,
[Name]

---
P.S. Assets (Screenshots, Pressemappe) kann ich sofort zusenden.
```

```

→ [Headline-Finding mit Zahl]

- [Finding 3]
- [Finding 4]

zur Verfügung stellen - gerne auch ein Gespräch mit
[Experte/Analyst] für Einordnung.

Interesse?

[Name]
```

## Personalisierungs-Bausteine

### Artikel-Referenzen

```
Standard:
"Ihr Artikel über [Thema] hat mich angesprochen, weil..."

Spezifisch:
"In Ihrem Artikel vom [Datum] erwähnten Sie, dass [Punkt].
Das deckt sich mit dem, was wir sehen..."

Kritisch:
"Ich war nicht 100% einverstanden mit [Punkt] - aber genau
deshalb könnten unsere Daten interessant sein..."
```

### Beat-Verbindungen

```
"Als jemand, der [Beat] so eng verfolgt..."
"Für Ihre [Beat]-Berichterstattung könnte das relevant sein..."
"Ich weiß, dass [Publikation] sich stark für [Beat] interessiert..."
```

### Timing-Hooks

```
"Passend zur aktuellen Diskussion über [Thema]..."
"Angesichts von [aktuelles Event/News]..."
"Mit Blick auf [Messe/Konferenz/Event]..."
```

### Gemeinsame Kontakte

```
"[Name] hat mir empfohlen, Sie zu kontaktieren..."
"Wir haben uns kurz auf [Event] unterhalten..."
"[Gemeinsamer Kontakt] meinte, das könnte Sie interessieren..."
```

## Anti-Patterns (vermeiden!)

### ❌ Generic Opening

```
❌ "Ich hoffe, es geht Ihnen gut."
❌ "Ich wollte mich kurz vorstellen..."
❌ "Wir sind ein führendes Unternehmen..."
```

### ❌ Wall of Text

```
❌ Mehr als 150 Wörter im ersten Pitch
❌ Keine Absätze/Struktur
❌ Alle Infos auf einmal
```

### ❌ Pushy Language

```
❌ "Das müssen Sie unbedingt bringen"
❌ "Ihre Leser werden das lieben"
❌ "Ich bin sicher, Sie interessiert das"
```

### ❌ Missing CTA

```
❌ "Lassen Sie mich wissen, was Sie denken"
❌ "Bei Fragen stehe ich zur Verfügung"
(Kein klarer nächster Schritt)
```

---

## Format-Regeln

### Länge

```
Subject: Max 50 Zeichen (für Mobile)
Email: 100-150 Wörter (scannable)
Absätze: Max 3 Zeilen
```

### Struktur

```
✅ Bullet Points für Key Facts
✅ Ein klarer CTA am Ende
✅ Signatur mit allen Kontaktdaten
✅ P.S. für zusätzlichen Hook (optional)
```

### Anhänge

```
Erster Pitch: Keine Anhänge
Follow-Up: Nur wenn angefragt
Ausnahme: Einbettbare Infografik-Vorschau
```
