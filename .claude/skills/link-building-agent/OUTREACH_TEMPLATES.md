# Link Building Outreach Templates

## Allgemeine Regeln

```
1. Kurz halten (max 150 Wörter)
2. Personalisieren (Name + spezifischer Bezug)
3. Mehrwert bieten (nicht nur nehmen)
4. Klarer CTA
5. Professionelle Signatur
```

---

## Guest Post Outreach

### Initial Pitch

```
Betreff: Gastartikel-Idee für [Publikation]

Hi [Name],

ich lese [Publikation] regelmäßig - besonders [spezifischer Artikel]
hat mir gefallen.

Ich würde gerne einen Gastartikel beisteuern:

"[Vorgeschlagener Titel]"

Gliederung:
• [Punkt 1]
• [Punkt 2]
• [Punkt 3]

Der Artikel würde ca. [X] Wörter haben mit praktischen
Beispielen und Daten.

Zu mir: [1 Satz Expertise]

Interesse?

Beste Grüße,
[Name]
```

### Follow-Up (nach 5 Tagen)

```
Betreff: Re: Gastartikel-Idee für [Publikation]

Hi [Name],

kurze Nachfrage zu meinem Pitch von letzter Woche.

Falls das Thema nicht passt, hätte ich Alternativen:
• [Alternative 1]
• [Alternative 2]

Ich orientiere mich gerne an Ihrem Redaktionsplan.

Beste Grüße,
[Name]
```

## Broken Link Building

### Initial Outreach

```
Betreff: Defekter Link auf [Seitenname]

Hi [Name],

ich bin auf Ihre Seite "[Seitentitel]" gestoßen -
tolle Ressourcensammlung!

Kleiner Hinweis: Der Link zu "[Beschreibung]" scheint
nicht mehr zu funktionieren:
[Defekte URL]

Falls Sie nach Ersatz suchen, haben wir einen
aktuellen Artikel zum Thema:
[Ihre URL]

Nur als Vorschlag - vielleicht hilfreich für Ihre Leser!

Beste Grüße,
[Name]
```

### Alternative (wenn kein Ersatz-Content)

```
Betreff: Defekter Link auf [Seitenname]

Hi [Name],

auf Ihrer Seite "[Titel]" ist mir ein defekter Link
aufgefallen:

[Defekte URL]

Dachte, das könnte Sie interessieren.

Falls Sie generell nach Ressourcen zu [Thema] suchen,
hätte ich ein paar Empfehlungen.

Beste Grüße,
[Name]
```

```
Betreff: Ressourcen-Vorschlag für [Seitenname]

Hi [Name],

ich habe Ihre Ressourcen-Seite zu [Thema] gefunden -
sehr hilfreich zusammengestellt!

Ich möchte [Ihren Artikel / Ihr Tool] vorschlagen,

[URL]

Es behandelt [kurze Beschreibung] und wird von
[Social Proof: X Nutzern / in Y Publikationen] genutzt.

Falls es passt, würde ich mich freuen. Wenn nicht,
kein Problem!

Beste Grüße,
[Name]
```

```

Hi [Name],

Ihre Ressourcen-Seite zu [Thema] ist eine der besten,
die ich gesehen habe.

   • [Ressource 1]
   • [Ressource 2]

Nur als Ideen - vielleicht nützlich für Ihre Leser!

Beste Grüße,
[Name]
```

## Unlinked Mention Outreach

```

mehr zu erfahren:
[URL]

Gerne revanchiere ich mich - haben Sie Content, den
wir in unseren Kanälen teilen könnten?

Beste Grüße,
[Name]
```

```
Sie [Marke] erwähnen.

Wir haben seitdem [Verbesserung] umgesetzt.]

Falls Sie die Erwähnung verlinken möchten, hier
unsere Website: [URL]

Ich stehe auch gerne für Fragen oder ein Update
zur Verfügung.

Beste Grüße,
[Name]
```

## Link Reclamation

### Verlorene Links zurückgewinnen

```
Betreff: Kurze Frage zu [Artikel-Titel]

Hi [Name],

ich habe bemerkt, dass der Link zu [unserer Seite]
in Ihrem Artikel "[Titel]" nicht mehr funktioniert
oder entfernt wurde.

War das Absicht oder vielleicht ein Versehen beim
Update?

Falls die Ressource noch relevant ist, hier die
aktuelle URL: [URL]

Falls nicht, verstehe ich natürlich. Feedback ist
auch willkommen!

Beste Grüße,
[Name]
```

```

Hi [Name],

kurze Nachfrage zu meiner Email von [Tag].

Ist das Thema interessant für Sie?

Beste Grüße,
[Name]
```

```

oder gibt es noch Interesse?

Kein Problem, falls nicht - ich respektiere Ihre Zeit.

Beste Grüße,
[Name]
```

```
Betreff: Re: [Original Subject]

Hi [Name],

verstanden - danke für die Rückmeldung!

Darf ich Sie für zukünftige Themen auf dem Radar
behalten? Ich melde mich gerne, wenn etwas
Passenderes kommt.

Beste Grüße,
[Name]
```

## Personalisierungs-Elemente

### Für Blogger

```
• Bezug auf spezifischen Artikel
• Ihre Expertise im Thema
• Gemeinsame Interessen (Social Media Check)
• Lokaler Bezug (falls relevant)
```

### Für Unternehmen

```
• Bezug auf Firmen-News
• Produkt/Service-Relevanz
• Branchentrends
• Gemeinsame Kontakte
```

### Für Journalisten

```
• Bezug auf kürzliche Artikel
• Beat-Relevanz
• Exklusivitäts-Angebot
• Daten/Statistiken als Hook
```

## Anti-Patterns

### ❌ Vermeiden

```
"Ich hoffe, es geht Ihnen gut"
"Ich möchte mich kurz vorstellen"
"Wir sind ein führendes Unternehmen"
"Bitte verlinken Sie zu uns"
"Das wäre toll für Ihr SEO"
"Backlink-Anfrage"
"Link Exchange"
```

### ✅ Stattdessen

```
Direkter Einstieg mit Value
Spezifischer Bezug zum Empfänger
Klarer Mehrwert für deren Leser
Natürliche Sprache
Professionell aber nicht steif
```
