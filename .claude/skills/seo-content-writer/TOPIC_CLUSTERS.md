# Topic Cluster Strategie

## Konzept

```
Topic Cluster = Pillar Page + Cluster Articles
              = Thematische Autorität aufbauen

Pillar Page:
- Breites Thema
- Umfassend
- Verlinkt zu allen Clustern

Cluster Articles:
- Spezifische Unterthemen
- Tiefgehend
- Verlinken zurück zum Pillar
```

## Pillar Page erstellen

### Themen-Auswahl

```
Gute Pillar-Themen:
✅ Hohes Suchvolumen (1.000+/Monat)
✅ Breites Thema mit vielen Unterthemen
✅ Wettbewerb ist schlagbar
✅ Relevant für dein Business
✅ Genug Content-Potenzial

Beispiele:
- "Content Marketing"
- "SEO für Anfänger"
- "Email Marketing"
- "Social Media Strategie"
```

### Pillar-Struktur

```markdown
# [Broad Topic]: Der ultimative Guide

## Inhaltsverzeichnis
(mit Sprungmarken)

## Was ist [Topic]?
→ Link zu: Cluster "Definition/Grundlagen"

## Warum [Topic] wichtig ist
→ Link zu: Cluster "Vorteile/Benefits"

## [Topic] Strategie entwickeln
→ Link zu: Cluster "Strategie"

## [Topic] in der Praxis
→ Link zu: Cluster "How-To"

## [Topic] Tools
→ Link zu: Cluster "Tools Vergleich"

## [Topic] Beispiele
→ Link zu: Cluster "Case Studies"

## Häufige Fehler bei [Topic]
→ Link zu: Cluster "Fehler vermeiden"

## [Topic] Trends & Zukunft
→ Link zu: Cluster "Trends"

## FAQ
(eigene FAQ + Links zu detaillierten Clustern)

## Fazit
```

## Cluster-Artikel planen

### Cluster-Mapping

```

1. Keyword-Recherche (Long-Tail)
2. People Also Ask (Google)
3. Related Searches (Google)
```

### Cluster-Priorisierung

```
Priorität 1 (zuerst erstellen):
- Hohes Suchvolumen
- Low Competition
- Direkt business-relevant

Priorität 2:
- Mittleres Suchvolumen
- Unterstützt Pillar

Priorität 3:
- Long-Tail / Nische
- Komplettiert das Cluster
```

## Verlinkung im Cluster

### Pillar → Cluster

```
Jeder Pillar verlinkt zu:
- ALLEN zugehörigen Clustern
- Im relevanten Kontext (nicht nur Liste am Ende)
- Mit beschreibendem Anchor-Text

Beispiel im Pillar:
"Eine durchdachte Content-Strategie ist die Grundlage
für erfolgreiches Content Marketing. In unserem Guide
zur [Content Strategie Entwicklung] zeigen wir dir
die einzelnen Schritte."
```

### Cluster → Pillar

```
Jeder Cluster verlinkt zum Pillar:
- Früh im Artikel (erste 200 Wörter)
- Kontextuell passend
- Nochmal im Fazit

Beispiel im Cluster:
"Content Strategie ist ein zentraler Baustein des
[Content Marketings]. Bevor du loslegst, solltest du
die Grundlagen verstehen."
```

### Cluster ↔ Cluster

```
Verwandte Cluster verlinken zueinander:
- Wenn thematisch sinnvoll
- Nicht erzwingen
- 1-2 Links pro Cluster zu anderen Clustern

Beispiel:
"Nachdem deine Strategie steht, geht es an die
Umsetzung. Unser Guide zum [Blog Artikel Schreiben]
hilft dir dabei."
```

## Topic Cluster aufbauen

### Phase 1: Pillar erstellen

```
Woche 1-2:
1. Keyword-Recherche für Pillar
2. Wettbewerber-Analyse
3. Outline erstellen
4. Pillar schreiben (3.000+ Wörter)
5. Platzhalter für Cluster-Links
```

### Phase 2: Priority Cluster

```
Woche 3-6:
1. Top 5 Cluster nach Priorität erstellen
2. Je 1.500-2.500 Wörter
3. Pillar-Links einbauen
4. Pillar aktualisieren (Links zu neuen Clustern)
```

### Phase 3: Cluster vervollständigen

```
Woche 7+:
1. Weitere Cluster erstellen
2. Cluster untereinander verlinken
3. Pillar kontinuierlich updaten
4. Neue Cluster-Ideen identifizieren
```

### Phase 4: Maintenance

```
Ongoing:
- Pillar alle 3-6 Monate updaten
- Neue Cluster bei Bedarf
- Broken Links prüfen
- Performance monitoren
```

## Content-Lücken finden

### Methode 1: Competitor Gap

```
1. Top 3 Wettbewerber für Pillar-Keyword
2. Deren Unterthemen analysieren
3. Was haben sie, was du nicht hast?
4. → Neue Cluster-Ideen
```

### Methode 2: Keyword Gap

```
1. Ahrefs/SEMrush Keyword Gap
2. Keywords, für die Wettbewerber ranken
3. Du nicht
4. → Cluster-Kandidaten
```

### Methode 3: Search Console

```
1. Search Console → Performance
2. Impressionen ohne Klicks
3. Diese Keywords brauchen Content
4. → Cluster-Ideen
```

### Methode 4: People Also Ask

```
1. Google Suche für Pillar-Keyword
2. "Ähnliche Fragen" notieren
3. Jede Frage = potenzieller Cluster
```

```

□ Pillar rankt für Hauptkeyword?

□ Content noch aktuell?
□ Neue Cluster-Opportunities?
```

## Häufige Fehler

### ❌ Zu breiter Pillar

```
Schlecht: "Digital Marketing" (zu breit)
Besser:   "Content Marketing" (fokussierter)
```

### ❌ Cluster ohne Pillar-Link

```
Jeder Cluster MUSS zum Pillar verlinken!
Sonst kein Cluster-Effekt.
```

### ❌ Überlappende Cluster

```
Schlecht:
- "Blog Artikel schreiben"
- "Bloggen für Anfänger"
(zu ähnlich → Keyword Cannibalization)

Besser: Zusammenlegen oder klar differenzieren
```

### ❌ Cluster zu kurz

```
Cluster sollten 1.500+ Wörter haben.
Unter 1.000 = meist zu dünn für Ranking.
```

### ❌ Pillar nicht updaten

```
Pillar muss "leben":
- Neue Cluster hinzufügen
- Veraltete Infos aktualisieren
- Datum aktualisieren
```
