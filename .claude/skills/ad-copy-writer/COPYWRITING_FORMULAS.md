# Copywriting Formulas für Ads

## Die wichtigsten Formeln

### 1. AIDA (Klassiker)

**A**ttention → **I**nterest → **D**esire → **A**ction

```
Attention: Frustriert von langsamen Reports?
Interest:  Mit unserem Tool erstellst du Reports in 5 Minuten statt 5 Stunden.
Desire:    Stell dir vor, wie viel Zeit du für wichtigere Aufgaben hättest.
Action:    Jetzt kostenlos testen →
```

**Beste für:** Awareness & Consideration Campaigns

---

### 2. PAS (Problem-fokussiert)

**P**roblem → **A**gitate → **S**olve

```
Problem:  Dein Content erreicht niemanden?
Agitate:  Jeden Tag verlierst du potenzielle Kunden an die Konkurrenz.
          Während andere wachsen, stagnierst du.
Solve:    Mit [Produkt] erreichst du 3x mehr Zielgruppe. Garantiert.
```

**Beste für:** Pain-Point-fokussierte Audiences

---

### 3. BAB (Transformation)

**B**efore → **A**fter → **B**ridge

```
Before:  Stundenlanges Erstellen von Social Media Posts.
After:   30 Posts in 10 Minuten - fertig für den ganzen Monat.
Bridge:  [Produkt] macht's möglich. Starte jetzt kostenlos.
```

**Beste für:** Produkte mit klarer Transformation

---

### 4. FAB (Feature-fokussiert)

**F**eature → **A**dvantage → **B**enefit

```
Feature:   KI-gestützte Content-Generierung
Advantage: Erstelle Inhalte 10x schneller
Benefit:   Mehr Zeit für Strategie, weniger für Produktion
```

**Beste für:** Tech-Produkte, Feature-Launches

---

### 5. 4 Ps

**P**icture → **P**romise → **P**roof → **P**ush

```
Picture:  Stell dir vor, dein Team arbeitet doppelt so produktiv.
Promise:  Mit [Produkt] ist das möglich - in nur 2 Wochen.
Proof:    500+ Unternehmen haben ihre Produktivität um 127% gesteigert.
Push:     Starte jetzt deine 14-tägige Testphase.
```

**Beste für:** B2B, höherpreisige Produkte

---

### 6. ACCA

**A**wareness → **C**omprehension → **C**onviction → **A**ction

```
Awareness:    73% der Marketing-Teams verschwenden Zeit mit manuellen Tasks.
Comprehension: Automatisierung kann 15 Stunden pro Woche sparen.
Conviction:   [Produkt] hat bereits 2.000 Teams geholfen.
Action:       Hol dir deine 15 Stunden zurück. Demo buchen →
```

**Beste für:** Edukativer Verkaufsansatz

---

### 7. QUEST

**Q**ualify → **U**nderstand → **E**ducate → **S**timulate → **T**ransition

```
Qualify:    Für Marketing-Manager mit 5+ Team-Mitgliedern
Understand: Du weißt, wie zeitaufwändig Reporting sein kann.
Educate:    Automatisierte Dashboards aktualisieren sich selbst.
Stimulate:  Dein Chef ist beeindruckt. Dein Team ist entlastet.
Transition: Teste es 14 Tage kostenlos →
```

**Beste für:** Spezifische Zielgruppen-Segmente

## Kurzformeln für Headlines

### The "How To"
```
Wie du [Ergebnis] erreichst (ohne [unerwünschtes])
→ "Wie du 10kg abnimmst (ohne Diät)"
→ "Wie du mehr Leads bekommst (ohne Kaltakquise)"
```

### The Number
```
[Zahl] [Dinge] für [Ergebnis]
→ "7 Tools für bessere Produktivität"
→ "3 Fehler die dein Marketing ruinieren"
```

### The Question
```
[Frage die Pain anspricht]?
→ "Frustriert von niedrigen Conversions?"
→ "Warum wächst deine Konkurrenz schneller?"
```

### The Command
```
[Imperativ] + [Benefit]
→ "Steigere deinen Umsatz um 47%"
→ "Spare 10 Stunden pro Woche"
```

### The Curiosity Gap
```
[Statement das neugierig macht]
→ "Das wissen die wenigsten über SEO..."
→ "Der Fehler, den 90% der Marketer machen"
```

### The Social Proof
```
[Zahl]+ [Zielgruppe] [Ergebnis]
→ "2.000+ Unternehmen steigern so ihre Leads"
→ "Von 500+ Marketing-Teams empfohlen"
```

## Formeln für Descriptions

### The Bridge
```
[Problem]? [Produkt] ist die Lösung. [Benefit]. [CTA].
→ "Content-Erstellung dauert ewig? ContentAI schreibt für dich. 10x schneller. Jetzt testen."
```

### The List
```
✓ [Benefit 1] ✓ [Benefit 2] ✓ [Benefit 3]. [CTA].
→ "✓ Mehr Leads ✓ Weniger Aufwand ✓ Bessere Ergebnisse. Kostenlos starten."
```

### The Proof
```
[Zahl] [Ergebnis] für [Zahl] [Kunden]. [Einladung].
→ "47% mehr Conversions für 500+ Shops. Dein Shop ist als nächstes dran."
```

### The Risk Reversal
```
[Angebot]. [Garantie]. [CTA].
→ "14 Tage kostenlos. Keine Kreditkarte. Jederzeit kündbar. Jetzt starten."
```

## Power Words für Ads

### Urgency
```
Jetzt, Heute, Sofort, Letzte Chance, Nur noch [X],
Endet bald, Begrenzt, Exklusiv, Neu
```

### Trust
```
Garantiert, Bewährt, Zertifiziert, Sicher, Offiziell,
Getestet, Empfohlen, Ausgezeichnet, #1
```

### Value
```
Kostenlos, Gratis, Geschenk, Bonus, Extra, Inklusive,
Sparen, Rabatt, Angebot, Deal
```

### Emotion
```
Endlich, Entdecke, Geheimnis, Verwandle, Revolutionär,
Überraschend, Unglaublich, Erstaunlich
```

### Exclusivity
```
Exklusiv, Premium, VIP, Nur für [Zielgruppe],
Insider, Früher Zugang, Limitiert
```

## Ad Copy Checkliste

### Vor dem Schreiben:
- [ ] Zielgruppe klar definiert?
- [ ] Kampagnenziel festgelegt?
- [ ] USP/Hauptbenefit identifiziert?
- [ ] Platform-Limits bekannt?

### Beim Schreiben:
- [ ] Hook in ersten 5 Wörtern?
- [ ] Benefits statt Features?
- [ ] Spezifisch statt vage?
- [ ] "Du" statt "Wir"?
- [ ] Aktive Sprache?
- [ ] Klarer CTA?

### Nach dem Schreiben:
- [ ] Zeichenlimits eingehalten?
- [ ] Mind. 3 Varianten erstellt?
- [ ] Compliance-Check?
- [ ] Korrektur gelesen?
