# Brand Visual Guidelines

## Markenfarben

### Primärfarben
| Farbe | HEX | RGB | Verwendung |
|-------|-----|-----|------------|
| Dark Blue | #003366 | 0, 51, 102 | Headlines, Primärelemente |
| Orange | #ff6600 | 255, 102, 0 | CTAs, Akzente, Highlights |

### Sekundärfarben
| Farbe | HEX | RGB | Verwendung |
|-------|-----|-----|------------|
| White | #ffffff | 255, 255, 255 | Hintergründe, Text |
| Dark Gray | #333333 | 51, 51, 51 | Body Text |
| Light Gray | #f5f5f5 | 245, 245, 245 | Hintergründe |

### Farbverhältnisse
```
60% - Neutral (Weiß, Grau)
30% - Primär (Dark Blue)
10% - Akzent (Orange)
```

## Bildstil

### Allgemeine Richtlinien

**DO:**
- Professionelle, hochwertige Bilder
- Natürliche Beleuchtung bevorzugen
- Authentische, ungestellte Momente
- Diversity in Personendarstellungen
- Moderne, cleane Ästhetik
- Positive, optimistische Ausstrahlung

**DON'T:**
- Überbearbeitete, künstliche Bilder
- Generische Stock-Photo-Ästhetik
- Überladene Kompositionen
- Veraltete Design-Trends
- Negative oder düstere Stimmung
- Klischeehafte Business-Posen

### Fotografie-Stil

#### Menschen
- Authentische Ausdrücke
- Business-casual bis professionell
- Diverse Altersgruppen, Ethnien, Geschlechter
- Natürliche Posen, keine "Stock-Photo-Gesten"
- Blickkontakt oder fokussierte Arbeit

#### Arbeitsumgebungen
- Moderne, helle Büros
- Clean Desks, minimaler Clutter
- Technologie sichtbar aber nicht dominant
- Pflanzen und natürliche Elemente
- Große Fenster, viel Tageslicht

#### Produkte/Services
- Sauberer Hintergrund (weiß oder gradient)
- Professionelle Produktfotografie
- Kontext-Shots in Verwendung
- Macro-Details für Features
- Konsistente Perspektiven

---

## Grafik-Design

### Layouts
- Asymmetrische Balance
- Viel Whitespace
- Klare visuelle Hierarchie
- Modulares Grid-System
- Links- oder rechtsbündiger Text-Bereich

### Typografie auf Bildern
- Headlines: Bold, groß, kontrastreich
- Subheadlines: Medium weight, kleiner
- Max 3 Textebenen pro Bild
- Text nie über komplexen Bildbereichen
- Immer ausreichend Kontrast

### Icons & Illustrationen
- Linien-Icons (nicht gefüllt)
- Konsistente Strichstärke
- Markenfarben verwenden
- Einfach und erkennbar
- Keine übermäßig detaillierten Illustrationen

## Plattform-spezifische Anpassungen

### LinkedIn (B2B, Professionell)
- Business-Fokus
- Daten/Charts wenn relevant
- Thought Leadership Ästhetik
- Dezente Markenfarben

### Instagram (Visuell, Lifestyle)
- Stärkere Farben
- Emotionalere Bilder
- Trend-bewusster
- Mehr Personality

### Facebook (Breit, Diverse Audience)
- Zugängliche Bildsprache
- Familiär aber professionell
- Community-Fokus
- Warme Töne

### TikTok (Jung, Dynamisch)
- Energetisch
- Authentisch > Poliert
- Trends aufgreifen
- Behind-the-Scenes OK

## Text-auf-Bild Guidelines

### Platzierung
```
┌─────────────────────┐
│ ████████            │  ← Headline oben links
│ ████████            │
│                     │
│                     │
│            ████████ │  ← CTA unten rechts
└─────────────────────┘
```

### Safe Zones für Text
- Mindestens 10% Abstand vom Rand
- Text nicht über Gesichtern
- Hintergrund unter Text: solid oder blur
- Textbereich: max 40% des Bildes

### Kontrast-Regeln
| Hintergrund | Textfarbe |
|-------------|-----------|
| Hell | #003366 (Dark Blue) |
| Dunkel | #ffffff (White) |
| Bild | Overlay + Weiß |
| Farbig | Weiß mit Schatten |

---

## Logo-Verwendung

### Platzierung
- Unten rechts (Standard)
- Unten links (Alternative)
- Nie in der Mitte
- Nie über wichtigen Bildelementen

### Größe
- Max 15% der Bildbreite
- Min erkennbar bei Thumbnail-Größe
- Konsistent über alle Assets

### Schutzraum
- Mindestens Logohöhe als Abstand zu allen Seiten
- Kein Text direkt am Logo

---

## Bild-Qualität

### Mindestanforderungen
| Verwendung | Min. Breite | DPI |
|------------|-------------|-----|
| Web | 1200px | 72 |
| Social | 1080px | 72 |
| Print | 300mm | 300 |
| Präsentation | 1920px | 72 |

### Dateiformate
| Format | Verwendung |
|--------|------------|
| JPG | Fotos, komplexe Bilder |
| PNG | Transparenz, Screenshots, Grafiken |
| SVG | Logos, Icons, Vektoren |
| WebP | Web-optimiert (mit Fallback) |

### Komprimierung
- JPG: 80-85% Qualität
- Max 200KB für Web-Bilder
- Progressive Loading aktivieren
- Alt-Text immer angeben
