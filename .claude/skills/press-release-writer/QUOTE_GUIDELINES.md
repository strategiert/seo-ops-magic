# Quote Guidelines für Pressemeldungen

## Zweck von Zitaten

Zitate in Pressemeldungen dienen dazu:
- **Menschlichkeit** hinzuzufügen
- **Meinung/Vision** auszudrücken (was der neutrale Text nicht kann)
- **Autorität** zu verleihen
- **Journalisten** O-Töne zu liefern

---

## Wer sollte zitiert werden?

### Primäres Zitat (Pflicht)
- Geschäftsführer/CEO
- Gründer
- Relevanter C-Level (CMO bei Marketing-News, CTO bei Tech-News)

### Sekundäres Zitat (Optional)
- Partner/Kooperationspartner
- Kunde (mit Erlaubnis)
- Branchenexperte

---

## Struktur eines guten Zitats

### Formel: Context + Opinion + Forward
```
[Kontext zu der Ankündigung]. [Meinung/Vision]. [Ausblick/Einladung].
```

### Beispiel:
```
„Mit dieser Partnerschaft erreichen wir einen wichtigen Meilenstein.
Gemeinsam können wir Mittelständlern endlich die Tools geben, die
bisher nur Konzernen vorbehalten waren. Wir freuen uns auf die
Zusammenarbeit."
```

---

## Dos and Don'ts

### DO ✅

**Vision zeigen**
```
„Wir glauben, dass jedes Unternehmen Zugang zu professionellem
Content Marketing haben sollte – unabhängig von der Größe."
```

**Emotionen ausdrücken**
```
„Wir sind stolz darauf, diesen Meilenstein erreicht zu haben."
„Es freut uns, diese Partnerschaft bekannt zu geben."
```

**Mehrwert erklären**
```
„Für unsere Kunden bedeutet das: weniger Aufwand, bessere Ergebnisse."
```

**Zukunft ansprechen**
```
„Das ist erst der Anfang. Für 2024 planen wir..."
```

### DON'T ❌

**Fakten wiederholen**
```
❌ „Wir haben heute unser neues Produkt gelauncht."
   (Das steht schon im Lead)
```

**Werbesprache**
```
❌ „Unser revolutionäres Produkt ist das beste am Markt."
```

**Zu technisch**
```
❌ „Die API-Integration ermöglicht RESTful Endpoints mit OAuth 2.0."
```

**Zu lang**
```
❌ [Zitat über 4-5 Sätze]
```

## Zitat-Templates

### Produkt-Launch
```
„Mit [Produkt] adressieren wir ein Problem, das [Zielgruppe] schon
lange beschäftigt: [Problem]. Ab sofort können sie [Lösung/Benefit].
Wir sind überzeugt, dass dies [positive Auswirkung]."
```

### Partnerschaft
```
„[Partner] teilt unsere Vision von [gemeinsame Vision]. Durch die
Zusammenarbeit können wir unseren Kunden [Mehrwert] bieten.
Wir freuen uns auf die gemeinsame Zukunft."
```

### Milestone/Erfolg
```
„[X Kunden/Nutzer/Umsatz] zu erreichen zeigt, dass unser Ansatz
funktioniert. Dieser Erfolg gehört vor allem unseren Kunden und
unserem Team. Er motiviert uns, noch besser zu werden."
```

### Studie/Report
```
„Die Ergebnisse unserer Studie zeigen deutlich: [Key Finding].
Für [Zielgruppe] bedeutet das, dass [Implikation]. Wir empfehlen,
[Handlungsempfehlung]."
```

### Investment/Funding
```
„Dieses Investment ermöglicht es uns, [Wachstumsziel] schneller
zu erreichen. Wir werden in [Bereich] investieren, um [Nutzen für Kunden].
Wir danken [Investoren] für ihr Vertrauen."
```

## Attribution-Formate

### Standard (Deutsch)
```
„Zitat", sagt Thomas Müller, Geschäftsführer der TechCorp GmbH.
```

### Mit Unternehmensbeschreibung
```
„Zitat", erklärt Dr. Anna Schmidt, CEO von ContentAI, einem Anbieter
für KI-gestützte Content-Erstellung.
```

### Für Folgezitate derselben Person
```
Müller fügt hinzu: „Weiteres Zitat."
```

### Für zweite Person
```
Auch [Partner-Name] zeigt sich zufrieden: „Zitat vom Partner."
```

---

## Verben für Attribution

| Verb | Verwendung |
|------|------------|
| sagt | Neutral, Standard |
| erklärt | Bei Erklärungen, Details |
| betont | Bei wichtigen Punkten |
| fügt hinzu | Bei Ergänzungen |
| kommentiert | Bei Reaktionen auf etwas |
| freut sich | Bei positiven Nachrichten |

**Vermeiden:**
- behauptet (impliziert Zweifel)
- meint (zu vage)
- findet (zu casual)
- ist der Meinung (zu umständlich)

---

## Zitat-Länge

### Ideal: 2-3 Sätze
```
„Erster Satz mit Kontext. Zweiter Satz mit Meinung oder Vision.
Dritter Satz optional mit Ausblick."
```

### Minimum: 1 Satz
Nur wenn sehr prägnant und aussagekräftig.

### Maximum: 4 Sätze
Darüber hinaus aufteilen oder kürzen.

---

## Beispiele: Vorher/Nachher

### Vorher (schlecht)
```
„Wir haben heute unser neues Produkt ContentAI gelauncht. Es ist
das beste Tool für Content Marketing. Kunden können damit schneller
arbeiten. Es hat viele Features wie KI-Texte und Analytics."
```

### Nachher (gut)
```
„Seit Jahren kämpfen Marketing-Teams mit dem gleichen Problem: zu
wenig Zeit für zu viel Content. Mit ContentAI geben wir ihnen ein
Werkzeug, das diese Gleichung verändert. Wir freuen uns darauf zu
sehen, was unsere Kunden damit erreichen werden."
```

---

## Checkliste für Zitate

- [ ] Fügt das Zitat Mehrwert hinzu (keine Fakten-Wiederholung)?
- [ ] Ist die Person für das Thema relevant?
- [ ] Ist das Zitat 2-3 Sätze lang?
- [ ] Enthält es Meinung/Vision (nicht nur Fakten)?
- [ ] Ist die Attribution korrekt (Name, Titel, Firma)?
- [ ] Keine Werbesprache?
- [ ] Würde ein Journalist das Zitat verwenden?
