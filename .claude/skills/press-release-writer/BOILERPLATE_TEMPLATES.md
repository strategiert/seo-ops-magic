# Boilerplate Templates

## Was ist ein Boilerplate?

Der Boilerplate ist ein standardisierter "Über uns"-Absatz am Ende jeder Pressemeldung. Er wird einmal erstellt und in allen PMs verwendet (mit gelegentlichen Updates).

---

## Boilerplate-Struktur

```
### Über [Firmenname]

[Firmenname] [Verb] [Hauptprodukt/-service] für [Zielgruppe].
[Gründungsjahr und/oder Standort]. [1-2 Sätze zu Erfolgen/Reichweite].
[Optional: Mission oder Vision]. Mehr Informationen unter [Website].
```

---

## Template nach Unternehmensphase

### Startup (Pre-Revenue / Early Stage)
```
### Über [Firma]

[Firma] entwickelt [Produkt/Service] für [Zielgruppe]. Das [Jahr]
gegründete Unternehmen mit Sitz in [Stadt] hat es sich zur Mission
gemacht, [Mission]. Das Team besteht aus erfahrenen [Branche]-Experten
von [bekannte Firmen, falls relevant]. Mehr unter [website].
```

### Scale-up (Wachstumsphase)
```
### Über [Firma]

[Firma] ist ein [Kategorie]-Anbieter für [Zielgruppe]. Seit der Gründung
[Jahr] hat das Unternehmen [Zahl] Kunden in [Zahl] Ländern gewonnen.
[Optional: Key Investoren oder Auszeichnungen]. Mit Hauptsitz in [Stadt]
und [Zahl] Mitarbeitern unterstützt [Firma] [Zielgruppe] dabei, [Nutzen].
Mehr unter [website].
```

### Etabliertes Unternehmen
```
### Über [Firma]

[Firma] ist ein führender Anbieter von [Produkt/Service]. Mit über [Zahl]
Kunden, [Zahl] Mitarbeitern und einem Jahresumsatz von [Betrag] gehört
das Unternehmen zu den [Top X / wichtigsten Anbietern] in [Branche].
Gegründet [Jahr] in [Stadt], bedient [Firma] heute Kunden in [Regionen].
Mehr unter [website].
```

## Beispiele nach Branche

### SaaS/Software
```
### Über ContentAI

ContentAI ist eine KI-gestützte Content-Marketing-Plattform für B2B-Unternehmen.
Die 2021 in Berlin gegründete Firma unterstützt über 2.000 Marketing-Teams dabei,
professionellen Content in einem Bruchteil der Zeit zu erstellen. ContentAI wurde
von G2 als "Leader" im Bereich Content Creation ausgezeichnet. Mehr Informationen
unter www.contentai.de.
```

### Agentur/Dienstleister
```
### Über MarketingPro GmbH

Die MarketingPro GmbH ist eine Full-Service-Digitalagentur für mittelständische
Unternehmen. Seit 2015 hat das Münchner Team über 200 Projekte für Kunden wie
[Beispielkunden] umgesetzt. Der Fokus liegt auf Performance Marketing, Content
und Marketing Automation. Mehr unter www.marketingpro.de.
```

### E-Commerce
```
### Über ShopBrand

ShopBrand ist ein Online-Händler für [Produktkategorie] mit Sitz in Hamburg.
Das 2018 gegründete Unternehmen beliefert über 100.000 Kunden in Deutschland,
Österreich und der Schweiz. ShopBrand wurde mehrfach für exzellenten Kundenservice
ausgezeichnet (Trusted Shops: 4.9/5). Mehr unter www.shopbrand.de.
```

## Elemente zum Einbauen

### Pflicht
- [ ] Firmenname
- [ ] Was die Firma macht (1 Satz)
- [ ] Zielgruppe
- [ ] Website

### Optional aber empfohlen
- [ ] Gründungsjahr
- [ ] Standort/Hauptsitz
- [ ] Kundenzahl oder Reichweite
- [ ] Mitarbeiterzahl (wenn beeindruckend)
- [ ] Bekannte Kunden (mit Erlaubnis)
- [ ] Auszeichnungen
- [ ] Investoren (bei Startups)

### Vermeiden
- Übertriebene Superlative ("der weltweit führende...")
- Buzzword-Bingo ("disruptiv", "revolutionär")
- Veraltete Informationen
- Interne Abkürzungen

---

## Boilerplate-Varianten

Manchmal braucht man verschiedene Versionen:

### Kurz (50-70 Wörter)
Für kurze PMs oder wenn Platz begrenzt.

### Standard (80-100 Wörter)
Für die meisten Pressemeldungen.

### Lang (120-150 Wörter)
Für wichtige Ankündigungen oder wenn mehr Kontext nötig.

### Englisch
Für internationale Medien.

---

## Pflege des Boilerplates

### Wann aktualisieren
- Nach signifikanten Milestones (10.000 → 25.000 Kunden)
- Nach Funding-Runden
- Bei Strategieänderungen
- Bei neuen wichtigen Auszeichnungen
- Mindestens 1x pro Jahr prüfen

### Versions-Kontrolle
Boilerplate mit Datum versehen und zentral speichern:
```
boilerplate_2024-01_de.md
boilerplate_2024-01_en.md
boilerplate_2024-01_kurz_de.md
```

## Pressekontakt-Block

Direkt nach dem Boilerplate:

```
### Pressekontakt

[Vollständiger Name]
[Position, z.B. "Head of Communications"]
[Firmenname]
E-Mail: [email]
Telefon: [telefon]
```

Oder kompakter:
```
### Pressekontakt
[Name] | [email] | [telefon]
```
