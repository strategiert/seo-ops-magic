# AP Style Guide für Pressemeldungen

## Grundprinzipien

### Objektivität
- Fakten, keine Meinungen
- Keine Superlative ohne Beleg
- Neutrale Sprache

### Klarheit
- Kurze Sätze (max 25-30 Wörter)
- Aktive Sprache
- Keine Fachbegriffe ohne Erklärung

### Präzision
- Genaue Zahlen
- Vollständige Namen
- Korrekte Titel

---

## Überschriften

### Regeln
- Präsens verwenden
- Artikel weglassen ("Firma startet" nicht "Die Firma startet")
- Keine Satzzeichen am Ende (außer Fragezeichen)
- Keine Abkürzungen (außer bekannte wie USA, EU)
- Zahlen als Ziffern (auch unter 10)

### Beispiele
```
✅ TechCorp startet KI-Plattform für Mittelstand
✅ Studie zeigt: 73% setzen auf Automatisierung
✅ 5 neue Partner verstärken Ökosystem

❌ Die TechCorp GmbH hat eine neue KI-Plattform gestartet.
❌ Neue Studie zeigt interessante Ergebnisse!
```

---

## Dateline

**Format:** `[STADT], [Tag]. [Monat] [Jahr] –`

```
BERLIN, 15. Januar 2024 –
MÜNCHEN, 3. März 2024 –
```

- Stadt in GROSSBUCHSTABEN
- Deutscher Datumsformat
- Gedankenstrich (–) nicht Bindestrich (-)

---

## Zahlen

### Grundregel
- 1-9: Ausschreiben (eins, zwei, drei...)
- 10+: Ziffern (10, 25, 100)

### Ausnahmen (immer Ziffern)
- Prozente: 5%, 73%
- Geldbeträge: 5 Euro, 1,5 Mio. Euro
- Alter: 5 Jahre alt
- Daten: 5. März
- Uhrzeiten: 9 Uhr, 14:30 Uhr

### Große Zahlen
```
1.000 (Tausend)
10.000
100.000
1 Mio. (Million)
1 Mrd. (Milliarde)
1,5 Mio. Euro
```

---

## Firmennamen

### Erste Erwähnung
Vollständiger Name mit Rechtsform:
```
TechCorp GmbH
ContentAI AG
Marketing Solutions GmbH & Co. KG
```

### Weitere Erwähnungen
Kurzform ohne Rechtsform:
```
TechCorp
ContentAI
Marketing Solutions
```

### Abkürzungen
Nur bei sehr bekannten Firmen. Bei unbekannten:
```
Erste Erwähnung: "Die Gesellschaft für Content Marketing (GCM)..."
Danach: "Die GCM..."
```

---

## Personen

### Erste Erwähnung
Voller Name + Titel/Position:
```
Thomas Müller, Geschäftsführer der TechCorp GmbH
Dr. Anna Schmidt, Chief Marketing Officer bei ContentAI
```

### Weitere Erwähnungen
Nur Nachname:
```
Müller erklärte weiter...
Schmidt betonte...
```

### Titel
- Akademische Titel: Dr., Prof. Dr.
- Keine Anreden (Herr, Frau) in Pressemeldungen
- Jobtitel auf Deutsch oder Englisch konsistent

## Anführungszeichen

### Deutsche Anführungszeichen
```
„Zitat hier einfügen", sagte Name.
```

### Zitat-Attribution
```
„Das ist ein Zitat", sagt Thomas Müller, Geschäftsführer der TechCorp GmbH.
„Das ist ein Zitat", erklärt Dr. Anna Schmidt.

Nicht:
„Das ist ein Zitat." sagt Thomas Müller.  ← Punkt falsch
"Das ist ein Zitat", sagt Thomas Müller.  ← Englische Anführungszeichen
```

### Verben für Attribution
```
sagt (neutral, Standard)
erklärt (bei Erklärungen)
betont (bei Wichtigem)
fügt hinzu (bei Ergänzungen)

Vermeiden:
behauptet (klingt zweifelnd)
meint (zu vage)
```

## Abkürzungen

### Erlaubt ohne Erklärung
```
USA, EU, UN, NATO
GmbH, AG, KG
CEO, CFO, CTO (bei Fachpublikum)
B2B, B2C (bei Fachpublikum)
```

### Mit Erklärung bei erster Verwendung
```
Die Künstliche Intelligenz (KI) ermöglicht...
Der Return on Investment (ROI) stieg...
```

## Typische Fehler vermeiden

### Werbesprache
```
❌ "Die revolutionäre neue Lösung..."
❌ "Der führende Anbieter..."
❌ "Das einzigartige Produkt..."

✅ "Die neue Lösung..."
✅ "Ein Anbieter von..."
✅ "Das Produkt..."
```

### Superlative
```
❌ "Der beste Service"
✅ "Ein Service mit [spezifischem Merkmal]"

❌ "Marktführer" (ohne Beleg)
✅ "Mit einem Marktanteil von 35%..."
```

### Passiv
```
❌ "Es wurde bekannt gegeben..."
✅ "TechCorp gab bekannt..."

❌ "Das Produkt wird angeboten..."
✅ "TechCorp bietet das Produkt an..."
```

## Deutsche vs. Englische Begriffe

### Deutsch bevorzugen
```
Geschäftsführer statt CEO (außer bei internationalen Firmen)
Pressemeldung statt Press Release
Neuigkeit statt News
Softwarelösung statt Software Solution
```

### Englisch akzeptabel
```
Marketing (kein gutes deutsches Äquivalent)
Content (etabliert)
Online (etabliert)
Website (etabliert)
E-Mail (mit Bindestrich)
```

### Hybrid vermeiden
```
❌ "Marketing-Aktivitäten performen gut"
✅ "Marketing-Aktivitäten zeigen gute Ergebnisse"
```
