# Pressemeldungs-Typen

## Übersicht

| Typ | Anlass | News-Wert |
|-----|--------|-----------|
| Produkt-Launch | Neue Produkte/Features | Hoch |
| Partnerschaft | Kooperationen | Mittel-Hoch |
| Milestone | Erreichte Ziele | Mittel |
| Studie/Report | Neue Erkenntnisse | Hoch |
| Event | Veranstaltungen | Mittel |
| Personalie | Neue Führungskräfte | Mittel |
| Award | Auszeichnungen | Mittel |
| Funding | Investitionen | Hoch |
| Expansion | Neue Märkte/Standorte | Mittel-Hoch |
| Krise | Reaktion auf Probleme | Situativ |

---

## 1. Produkt-Launch

### Wann nutzen
- Neues Produkt
- Signifikantes Feature-Update
- Neue Produktlinie

### Struktur
```
HEADLINE: [Firma] stellt [Produkt] vor: [Hauptbenefit]

LEAD: [Firma] hat heute [Produkt] vorgestellt, [eine Lösung für Zielgruppe].
[Key Benefit in einem Satz].

BODY 1: Was das Produkt macht (Features → Benefits)
BODY 2: Warum jetzt / Marktkontext
QUOTE: Vision des Gründers/CEO
BODY 3: Verfügbarkeit, Preise
QUOTE 2: Optional: Early Adopter/Beta-Kunde
```

### Beispiel-Headline
```
ContentAI stellt KI-Content-Plattform vor: Professioneller Content in Minuten
```

---

## 2. Partnerschaft

### Wann nutzen
- Strategische Kooperation
- Technologie-Integration
- Vertriebspartnerschaft
- Joint Venture

### Struktur
```
HEADLINE: [Firma A] und [Firma B] [kooperieren/partnern] für [Ziel]

LEAD: [Firma A] und [Firma B] haben eine [Art der Partnerschaft] bekannt gegeben.
[Was das für Kunden bedeutet].

BODY 1: Details der Partnerschaft
BODY 2: Was jeder Partner einbringt
QUOTE 1: CEO/Gründer Firma A
QUOTE 2: CEO/Gründer Firma B
BODY 3: Nächste Schritte, Verfügbarkeit
```

### Beispiel-Headline
```
ContentAI und HubSpot kooperieren für nahtlose Marketing-Automation
```

---

## 3. Milestone/Erfolg

### Wann nutzen
- Kundenzahl erreicht
- Umsatzziel erreicht
- Nutzungsstatistik
- Jubiläum

### Struktur
```
HEADLINE: [Firma] erreicht [Milestone]: [Bedeutung]

LEAD: [Firma] hat [Milestone] erreicht. [Was das über den Erfolg aussagt].

BODY 1: Details zum Milestone
BODY 2: Weg zum Milestone (kurz)
QUOTE: Dank an Kunden/Team + Ausblick
BODY 3: Was als Nächstes kommt
```

### Beispiel-Headline
```
ContentAI überschreitet 10.000 Kunden: Verdopplung in 12 Monaten
```

---

## 4. Studie/Report

### Wann nutzen
- Eigene Marktforschung
- Branchen-Report
- Umfrage-Ergebnisse
- Trend-Analyse

### Struktur
```
HEADLINE: [Studie/Report]: [Key Finding mit Zahl]

LEAD: Eine neue [Studie/Umfrage] von [Firma] zeigt: [Haupterkenntnis].
[Implikation für die Branche].

BODY 1: Methodik kurz (N=X, Zeitraum)
BODY 2: Top 3 Findings
BODY 3: Was das bedeutet
QUOTE: Einordnung durch Experten
BODY 4: Wo man den Report bekommt
```

### Beispiel-Headline
```
Studie: 73% der Marketing-Teams setzen 2024 auf KI-Tools
```

---

## 5. Event-Ankündigung

### Wann nutzen
- Konferenz
- Webinar
- Produkt-Demo
- Kunden-Event

### Struktur
```
HEADLINE: [Firma] lädt ein zu [Event]: [Thema/Mehrwert]

LEAD: [Firma] veranstaltet am [Datum] [Event] zu [Thema].
[Was Teilnehmer erwartet].

BODY 1: Details (Datum, Ort, Programm)
BODY 2: Speaker/Highlights
QUOTE: Warum dieses Event wichtig ist
BODY 3: Anmeldung/Tickets
```

---

## 6. Personalie

### Wann nutzen
- Neue C-Level Position
- Neuer Beirat/Vorstand
- Signifikante Beförderung

### Struktur
```
HEADLINE: [Name] wird [Position] bei [Firma]

LEAD: [Firma] hat [Name] als neue(n) [Position] bekannt gegeben.
[Kurze Relevanz/Hintergrund].

BODY 1: Hintergrund der Person
BODY 2: Was sie bei der Firma machen wird
QUOTE: CEO über die Personalie
QUOTE 2: Die neue Person selbst
```

---

## 7. Award/Auszeichnung

### Wann nutzen
- Branchenpreis
- Zertifizierung
- Ranking-Platzierung
- Kunden-Bewertung (z.B. G2)

### Struktur
```
HEADLINE: [Firma] gewinnt [Award] für [Kategorie/Leistung]

LEAD: [Firma] wurde mit dem [Award] ausgezeichnet.
[Bedeutung des Awards].

BODY 1: Was der Award würdigt
BODY 2: Kriterien/Jury
QUOTE: Dank + was das für Kunden bedeutet
```

---

## 8. Funding/Investment

### Wann nutzen
- Seed/Series A/B/C
- Strategischer Investor
- IPO-Ankündigung

### Struktur
```
HEADLINE: [Firma] erhält [Betrag] in [Runde] – [Verwendung]

LEAD: [Firma] hat eine [Runde] über [Betrag] abgeschlossen.
Angeführt von [Lead-Investor]. [Wofür das Geld verwendet wird].

BODY 1: Details zur Runde (Investoren, Bewertung wenn öffentlich)
BODY 2: Verwendung der Mittel
QUOTE: CEO über Pläne
QUOTE 2: Lead-Investor über Gründe für Investment
BODY 3: Bisherige Erfolge
```

---

## 9. Expansion

### Wann nutzen
- Neuer Markt (Land/Region)
- Neuer Standort/Büro
- Neue Abteilung/Team

### Struktur
```
HEADLINE: [Firma] expandiert nach [Markt/Ort]: [Ziel]

LEAD: [Firma] hat die Expansion nach [Markt] bekannt gegeben.
[Warum dieser Markt].

BODY 1: Details zur Expansion
BODY 2: Was das für Kunden bedeutet
QUOTE: Vision für den Markt
BODY 3: Team/Ansprechpartner vor Ort
```

---

## News-Wert Checkliste

Bevor eine Pressemeldung geschrieben wird, prüfen:

- [ ] Ist das wirklich eine Neuigkeit?
- [ ] Interessiert das jemanden außerhalb der Firma?
- [ ] Gibt es einen konkreten Anlass (nicht nur "wir wollen PR")?
- [ ] Können wir Zahlen/Fakten nennen?
- [ ] Ist der Zeitpunkt richtig?

**Wenn 3+ mit "Nein" → Überdenken ob PM nötig ist.**
