# CTA Library - Call-to-Actions

## Übersicht nach Ziel

| Ziel | CTA-Typ | Beispiele |
|------|---------|-----------|
| Engagement | Frage, Poll | "Was denkt ihr?", "Stimmt ab!" |
| Traffic | Link-CTA | "Link in Bio", "Mehr im Blog" |
| Leads | Download | "Kostenloser Guide", "Checkliste" |
| Sales | Direkt | "Jetzt entdecken", "Demo buchen" |
| Community | Follow | "Folgt für mehr", "Newsletter" |

---

## Engagement CTAs

### Fragen (Kommentare generieren)
```
Was ist eure Erfahrung damit?
Wie handhabt ihr das in eurem Unternehmen?
Stimmt ihr zu oder seht ihr das anders?
Was würdet ihr hinzufügen?
Welcher Punkt resoniert am meisten?
Was ist euer größtes Learning in diesem Bereich?
Kennt ihr das Gefühl?
Habt ihr ähnliche Erfahrungen gemacht?
Was fehlt in dieser Liste?
Wie macht ihr das?
```

### Abstimmungen (Polls)
```
Option A oder Option B?
Team [X] oder Team [Y]?
Welches Problem ist größer: A / B / C?
Was kommt zuerst: [Option 1] oder [Option 2]?
```

### Save/Bookmark
```
Speichert euch das für später!
Bookmark für dein nächstes Projekt.
Save this for your next campaign.
Pin das für später.
```

### Share
```
Teilt das mit jemandem, der das braucht.
Tag someone who needs to hear this.
Kennt ihr jemanden, dem das hilft?
Leite das an dein Team weiter.
```

```

Weiterlesen: [Link]

```

## CTA-Formeln

### The Question Close
```
[Content]

Was denkst du? Schreib's in die Kommentare.
```

### The Either/Or
```
[Content]

Bist du Team A oder Team B?
```

### The Tease
```
[Teil 1 des Contents]

Willst du den Rest wissen?
Folge für Part 2.
```

### The Resource
```
[Content]

Ich habe eine kostenlose Checkliste
dazu erstellt. Link in Bio.
```

### The Personal Invite
```
[Content]

Wenn du Fragen hast, schreib mir
eine DM. Ich antworte auf jede.
```

## CTA Timing

| Content-Typ | CTA-Platzierung |
|-------------|-----------------|
| Educational | Am Ende nach Mehrwert |
| Story | Nach der Pointe |
| Carousel | Letzte Slide |
| Video | Verbal + Text am Ende |
| Thread | Letzter Tweet |

## Don'ts

- Zu viele CTAs in einem Post
- CTA vor dem Mehrwert
- Aggressive Sales-CTAs ohne Kontext
- "Click here" ohne Begründung
- Generische CTAs ohne Persönlichkeit
